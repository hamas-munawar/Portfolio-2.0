## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- GSAP for animations
- Particles.js for interactive backgrounds

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
